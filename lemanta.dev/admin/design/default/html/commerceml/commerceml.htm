    Справка
    При настройке 1С необходимо сообщить показанный выше URL обмена - это {**} адрес страницы синхронизации, через которую 1С будет обмениваться информацией {**} с сайтом магазина. В качестве логина и пароля для 1С необходимо сообщить {**} логин/пароль любого администратора, в правах которого разрешен доступ к странице {**} этого модуля. Даже рекомендуется создать отдельного {**} администратора и назначить ему права доступа - только {$module_pointer|escape}.
    Вы также можете использовать произвольный URL обмена, но в этом случае нужно {**} исправить в корневом файле .htaccess {**} стандартный относительный адрес cml (он назван там точкой синхронизации {**} 1С и Мой Склад) на выбранный вами адрес, например my-secret/very-secret/page.
    Только с IP. В этом поле необходимо перечислить через запятую IP-адреса {**} или их начальные части, чтобы ограничить посторонним доступ к функциям синхронизации. {**} Например, 22.32.42.52, 33.73, 11.12.13.14 - тогда синхронизацию смогут делать {**} только те компьютеры, чей адрес начинается с одного из перечисленных в списке примеров {**} (это два компьютера 22.32.42.52 и 11.12.13.14 и вся подсеть 33.73.*.*). {**} Пустое поле означает неограниченный доступ с любого адреса.
    Хранить файлы. Данный флажок дает возможность не удалять сразу же {**} после обработки файлы импорта, полученные в процессе синхронизации, а хранить их в папке {**} {$smarty.const.FOLDERNAME_FOR_ENGINE_OBJECTS|default:''|escape}/{$module_pointer|lower|escape}/{$upload_folder|default:''|escape} {**} для дальнейшего исследования. Предполагается, что флажок будет включаться только {**} на время отладки, когда необходимо узнать, в какой форме приходят сведения из 1C. {**} В остальное время флажок рекомендуется отключить, так как хранимые файлы будут {**} занимать все больше и больше файлового пространства сайта.
